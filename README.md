# framework-cg

Framework de ETL em Python criado para automatizar pipelines com PostgreSQL, Pandas e logging estruturado.

## Instalação
```bash
pip install "git+https://github.com/<teu-usuario>/framework-cg.git"
```

## Uso

```bash
from framework_cg import ProcessLogger, PostgresConnection, Extract, DataLoader, Transformer
```
